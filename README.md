  ## AI Chatbot Setup

This website includes an AI chatbot powered by **Gemini 2.5 Flash**.

### Setup Instructions

1. Get your Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey)

2. Create a `.env` file in the root directory:

```bash
VITE_GEMINI_API_KEY=your_api_key_here
```

3. The chatbot will appear as a floating button in the bottom-right corner of the website.
